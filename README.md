# TetrioDestroyer
A Tetris bot that plays on jstris with the pure intention of destroying the opponent.

# Jstris 
## Controls  
Move Left: a  
Move Right: d  
Soft Drop: s  
Hard Drop: w  
Rotate Left: q  
Rotate Right: e  
Rotate 180: r  
Hold: c  

## Game settings
Grid: None  
Ghost: Disabled
